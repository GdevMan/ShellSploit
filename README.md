# How to install
1. Install the file (or git clone the repo)
2. Install the requerments.txt file
3. run python shellsploit.py
## What to do if running/installing didn't work
1. Check if you are in the directory ```shellsploit```
2. Check if you installed requerments.txt (to install run ```pip install -r requerments.txt```
